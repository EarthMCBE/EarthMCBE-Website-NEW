# 服务器基础指令
## 基础指令
* /help  
  展示帮助菜单(含有英文)
* /me  
  在聊天中作出指定的动作
* /cordsearch  
  查看你在现实世界中在哪里
## 传输指令
* /tpa <玩家名>  
  请求传输到另一位玩家
* /accept  
  同意另一位玩家的传输请求
* /refuse  
  拒绝另一位玩家的传输请求
* /back  
  返回上一次传送地点(不记录死亡地点)
* /sethome  
  设置玩家当前位置为家的位置
* /gohome  
  返回设置的家
## 经济指令
* /mymoney  
  查看自己所拥有的钱
* /pay <玩家名>  
  向其他玩家付钱
